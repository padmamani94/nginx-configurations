<h1 align="center">Nginx Configurations</h1>

## Overview

This repository contains commonly used Nginx configuration examples for DevOps engineers. It includes configurations for reverse proxy, load balancing, SSL, and performance optimization.

## Configurations Included

- Reverse Proxy
- Load Balancer
- SSL Configuration
- Gzip Compression

## Prerequisites

- Ubuntu or Amazon Linux
- Nginx Installed
- Basic Linux Knowledge

## Test Configuration                        
sudo nginx -t


## Restart Nginx
sudo systemctl restart nginx


## Check Nginx Status
sudo systemctl status nginx


## Reload Configuration
sudo systemctl reload nginx
